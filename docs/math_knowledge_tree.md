# TeachAgent 数学知识树

这份树的目标是给后续 `rag`、`chunk`、`retriever` 和遗忘函数共用。

## 使用约定

- 叶子节点 = 最小可检索、可复习、可绑定错题的知识单元
- 父节点 = 只用于归类，不直接作为 chunk
- 错题绑定 = 1 个主节点 + 0~1 个辅节点
- 复习条目 = 叶子节点 + 典型易错点 + 1 个例题/变式题
